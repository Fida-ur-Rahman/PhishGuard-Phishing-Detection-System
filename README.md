# PhishGuard — Phishing Detection System

## Description

PhishGuard is a phishing detection system that analyzes URLs using machine learning to identify **safe** and **malicious** websites. It includes a simple web interface built with HTML, CSS, and JavaScript.

---

##  Features

* URL-based phishing detection
* Machine learning model
* Simple and user-friendly web interface

---

##  Tech Stack

* Python
* Scikit-learn
* HTML, CSS, JavaScript

---

##  Setup

```bash id="p0k3x1"
git clone https://github.com/your-username/PhishGuard.git
cd PhishGuard
pip install -r requirements.txt
python app.py
```

---

##  How It Works

User enters a URL → System analyzes it → Model predicts **Safe** or **Phishing**
